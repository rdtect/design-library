Initial release of the rdtect design library.

- `@rdtect/tokens` — 8 CSS token files (colors, elevation, blur, motion, typography, spacing, radius, z-index) following the Bronze Cathedral palette
- `@rdtect/ui` — Svelte 5 component library: primitives (Button, Card, Badge, Input) + patterns (GoldHeading, SurfaceCard, ColorizeReveal) + magnetic-button behavior
- `@rdtect/tailwind-preset` — Tailwind v4 preset bridging CSS custom properties to Tailwind utility classes
